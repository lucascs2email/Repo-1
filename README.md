
# SeekNow Flutter Client

Simple Flutter app for interacting with the SeekNow API.

## Features

- Universal search endpoint
- JSON result viewer
- Codemagic-ready structure
- Flutter Material 3 UI

## Setup

1. Install Flutter:
https://flutter.dev

2. Run dependencies:

```bash
flutter pub get
```

3. Start the app:

```bash
flutter run
```

## Important

Never hardcode your API key into public repositories.
Enter it at runtime or use environment variables/secrets.

## Codemagic

1. Push repo to GitHub
2. Connect repo to Codemagic
3. Build Android/iOS automatically
